Movie Dataset Sort

Overview

  • Team - Solo Project
  • Class - 355 Visual Analytics
  • Languages/Libraries - JavaScript, VegaLite, D3

For one of my courses related to data visualization, I had to create a notebook written in JavaScript that deals with datasets. My dataset was related to movies and I had to create visualization to investigate and answer some questions I came up with. I wanted to categorize movies by genre, but the issue was some movies contained multiple genres as shown below.

Problem

The dataset had 2000 movies in the list and I had to find a way to create a new list with only the specified genres I want. I also needed to create visualizations that answer my original investigative question of what factors affect box office success?

Process

Each movie contained a metadata of genre, if all movies had one genre then I can move on to creating visualizations, the problem is most movies contained multiple genres in string form.

The final result was a cleaned out dataset sorted by genre. I had some fun with it and explored some connections between ratings, genre and runtimes. Since this was a dataset with the top 100 movies of each year from 2002-2023, it was no surprise that the average rating was quite high regardless of genre.

Another connection I explored was between runtime and rating. Mapping out all the movie runtimes and their rating showed a positive correlation, with most movies falling around 100-120 minutes.

Reflection

Finding out hidden connections through tons of data and creating easily digestable diagrams was a fun exercise, this was a fun experience that deepened my JavaScript skills as well as learn more about data sorting and data visualization.
