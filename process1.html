YouTube Shorts Converter Extension

Overview

  • Team - Solo Project
  • Personal Project outside of school
  • Languages - JavaScript/HTML/CSS

Sometime around last year, YouTube introduced YouTube Shorts, a vertical video format that’s supposed to emulate popular video sites like Tik-Tok and Instagram Reels. The problem with shorts however, was that it was turning some short videos that were not meant to be shorts into YT Shorts. My issue with that is YT shorts gives you no control over playback, sound is only a toggle, and the video is fit into a vertical frame

Process

My solution to this was a coding challenge for me, I created a chromium plugin that can automatically convert YT Shorts back into the proper video format. YouTube videos have a unique id and what shorts does is change it from watch?v={uniqueID} to shorts?v={uniqueID}. The solution was quite simple, grab the URL, find the unique ID, then convert the link. The only issue was I’ve never worked with extensions and javascript before

After some research, I figured out that my first step should be to find the current tab the user is using. Then I do a check if the user is one a shorts video, if it is, then start the process of converting it. I separate the full link string and isolate the unique video ID.

From there, I create a new string that basically merges Youtube link + watch?v= + unique ID. Once that’s done, I created a function that automatically updates the current tab’s url to the fixed youtube link thereby getting rid of shorts. The actual UI needs some work but it’s fully functional and with a click of a button annoying youtube shorts are replaced with the real video player

Reflection

This was a fun experience for me to learn JavaScript and coding chrome extensions. The final product looks very basic and incomplete, but the extension works as intended so I'm satisfied.
