SFU Quizlet

Overview

  • Team - Pocholo Andres, John Ordoyo, Taha Liaqat
  • Class - 359 Mobile Computing
  • Languages - Kotlin, Java

SFU Quizlet was a final project for my Android Development class, I was in charge of creating and designing the front page as well as the course list. The app was written in Kotlin, and firebase was used for our database that contained all the courses, questions/answers, as well as user data.

Challenges

Designing the front page came with a lot of challenges, it was my first time working with an online-hosted database, and first time designing an app home page. One of the main issues I ran into was updating the front page in real time, the actual app loaded before the items from the database came in resulting in an empty home page. I had to devise a way to actively reload the page whenever any of the info listed gets changed.

Additionally, I had to design the front page to be clean and have all the most "important" information a user might need (like their favourite courses) or incentives that might help motivate the user (like stats about their cards practiced etc.)

In terms of designing the course list, it was relatively straight forward, we sorted the courses by department, and list the courses in sequential order. The users are able to bookmark specific courses that will appear on the top and on their home page.

Reflection

Overall, SFU Quizlet was a challenging but fun project that taught me the basics of how online databases work, in addition to further developing my Java/Kotlin development skills.
